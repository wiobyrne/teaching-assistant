# Which model and why

This project was built and tested around `qwen2.5:1.5b`.

## Why this model

It is small enough to be practical on a Raspberry Pi while still being useful for teacher-facing drafts, handouts, scaffolds, and profile review.

## Where to learn more

- Qwen official site: <https://qwen.ai/>
- Qwen GitHub organization: <https://github.com/QwenLM>
- Ollama library: <https://ollama.com/library/qwen2.5>

## What to expect

If you use a small local model on a Pi:

- responses may be slower than on a desktop
- shorter prompts usually work better
- keeping the profile simple helps the assistant stay useful

## Can you use another model?

Yes.

The repo uses `qwen2.5:1.5b` as the reference point, but teachers who have different hardware or different preferences can choose another model in Ollama.
